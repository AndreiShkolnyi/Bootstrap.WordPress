1. Нажмите зеленую кнопку Code 2. Далее Download ZIP 3. После загрузки распакуйте архив с проектом. 4. Скопируйте содерижмое архива на свой локальный сервер 5. (Запустить локальный сервер можно при помощи OpenServer для Windows, либо при помощи MAMP/MAMP PRO для Mac OS). 6. Дальше вам необходимо добавить '/wp-admin' в адресной строке после адреса локального сервера. 7. Логин и пароль для авторизации : 'login: admin ; password: 123456789'.
