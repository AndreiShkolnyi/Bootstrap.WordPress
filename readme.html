Нажмите зеленую кнопку Code Далее Download ZIP После загрузки распакуйте архив с проектом. Скопируйте содерижмое архива на свой локальный сервер (Запустить локальный сервер можно при помощи OpenServer для Windows, либо при помощи MAMP/MAMP PRO для Mac OS). Дальше вам необходимо добавить '/wp-admin' в адресной строке после адреса локального сервера. Логин и пароль для авторизации : 'login: admin ; password: 123456789'.
